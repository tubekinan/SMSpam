PROJECT: iOS SMS Spam Filter App

DESCRIPTION:
An iOS application that filters incoming SMS messages using Apple's Message Filter Extension. The app classifies messages as spam (junk) or allowed, prevents unnecessary notifications indirectly via system filtering, and provides a monitoring interface listing filtered messages.

TECH STACK:
- Swift
- Xcode
- IdentityLookup Framework (Message Filter Extension)
- App Groups (shared storage)
