## What is SMSpam?

SMSpam is an iOS application that helps you:

- Automatically detect spam SMS messages
- Block unwanted messages before they reach your inbox
- Manage a whitelist of trusted senders
- Customize spam detection rules
- Track blocked messages with detailed logs

## Key Features

| Feature | Description |
|---------|-------------|
| Spam Detection | Multiple detection methods |
| Whitelist | Trusted sender management |
| Regex Rules | Pattern-based filtering |
| Keyword Blocking | Word-based filtering |
| Short URL Detection | Suspicious link identification |
| Multi-language | 7 languages supported |
| Privacy First | All processing on-device |

## System Requirements

- iOS 16.0 or later
- iPhone, iPad compatible
- No internet connection required
